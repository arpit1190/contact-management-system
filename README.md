# Contact Management System

This is a full-stack Contact Management System built using:

- 🧠 C++ for backend logic
- 🌐 Node.js + Express.js for API server
- 🎨 HTML, CSS, JavaScript for frontend
- 📁 JSON for data storage

## 🚀 Features

- ✅ Add Contact
- 🔍 Search Contact
- 🗑️ Delete Contact
- 📃 Display All Contacts
- 💾 Stores data permanently in `data.json`

## 🛠 Technologies Used

- C++ (logic)
- Node.js & Express.js (backend)
- HTML, CSS, JavaScript (frontend)
- JSON file handling
- nlohmann/json.hpp (C++ JSON library)
